# 简介

- 前端通用方法工具库。

# 使用方法：

### 1. import：

```javascript
npm install q-tools;
import { mobileDesensitize } from 'q-tools';
const result = mobileDesensitize(15012345678);
```

### 2. script:

```javascript
<script src="q-tools.umd.js"></script>
<script>
  const result = QTools.mobileDesensitize(15012345678);
</script>
```
